# Olist E‑commerce Performance Report

An interactive data analytics project built with Streamlit and Plotly to explore the Brazilian Olist e‑commerce dataset.  
The app provides a comprehensive view of sales, delivery performance, and customer satisfaction through dynamic KPIs, charts, and maps.

## Project Goals

- Analyze Olist sales and revenue over time
- Understand delivery performance and on‑time delivery rate
- Explore how delivery time impacts customer review scores
- Visualize customer locations across Brazil
- Practice and showcase data analytics, visualization, and dashboard design skills

---

## Features

- **Global KPIs**
  - Total revenue
  - Total orders
  - Average review score
  - total customers, on‑time delivery rate

- **Time‑Series Analysis**
  - Monthly revenue trend
  - Comparison of revenue vs. number of orders over time

- **Customer Satisfaction**
  - Review score distribution

- **Geographical Analysis**
  - Customer location map aggregated by zip code / city / state
  - Filters by customer state

- **Interactive Filtering**
  - Filter by product category
  - Filter by customer state
  - All KPIs and charts update based on selected filters

---

## Tech Stack

- **Language:** Python 3.12
- **Web App:** [Streamlit](https://streamlit.io/)
- **Data Handling:** [Pandas](https://pandas.pydata.org/)
- **Visualization:** [Plotly Express](https://plotly.com/python/)
- **Styling:** Custom CSS injected into Streamlit (`styles.css`)
